# Find Yourself

`Find Yourself` 是一款面向求职者的 AI 求职伴侣。它把“聊聊、简历输入、分析优化、行动复盘”串成一个连续流程，帮助用户更快明确方向、优化材料，并把建议落实到行动。

## 产品定位

这不是单纯的简历工具，也不是一次性问答机器人。它更像一个围绕求职全流程工作的助手，核心目标是：

- 帮用户把求职困惑说清楚
- 帮用户看懂自己和岗位的差距
- 帮用户把分析结果转成可执行的下一步
- 帮用户持续复盘、修正和推进

## 当前页面

- `chat.html`：聊聊页，负责需求交流和对话
- `resume.html`：简历输入页，负责简历与目标 JD 的录入
- `analyze.html`：分析与优化页，负责方向、匹配、润色和面试模拟
- `journey.html`：行动与复盘页，负责今日行动、阶段进度和复盘记录
- `index.html`：首页入口

## 核心功能

### 1. 聊聊

- 通过开放式对话了解用户当前困惑、目标和限制
- 保存聊天记录、用户画像和记忆状态
- 支持悬浮聊天入口，方便在其它页面快速提问

### 2. 简历输入

- 输入或粘贴简历内容
- 输入目标岗位 JD
- 支持从 PDF / DOCX 提取文本
- 保存草稿，便于后续继续编辑

### 3. 分析与优化

页面上提供四个主要分析入口：

- 职业方向与行动计划
- 岗位匹配分析
- 简历润色
- 面试模拟

每个入口都对应独立的结果区，结果会以更自然的中文文本呈现。

### 4. 行动与复盘

- 记录今天最重要的行动项
- 显示当前求职阶段
- 保存阻碍、收获和下一步
- 展示最近的复盘痕迹

## 数据与记忆

系统会在浏览器本地保存一些状态，例如：

- 对话记录
- 草稿内容
- 用户画像
- 行动与复盘记录
- 情绪记录

后端也支持按 `client_id` 保存更完整的记忆数据，用于连续对话和画像更新。

## 技术栈

- Python + FastAPI
- 前端静态页面 + 原生 JavaScript
- SSE 流式聊天
- PDF / DOCX 文本提取
- Word 导出

## 本地运行

### Windows 一键启动

在项目根目录双击 `run.ps1`，或者在 PowerShell 中执行：

```powershell
.\run.ps1
```

脚本会自动：

- 创建 `.venv`（如果不存在）
- 安装依赖
- 启动服务

### 手动启动

1. 复制 `.env.example` 为 `.env`，并填写模型服务相关配置。
2. 安装依赖：

```bash
pip install -r requirements.txt
```

3. 启动服务：

```bash
uvicorn main:app --reload
```

4. 打开页面：

- `http://127.0.0.1:8000/`
- `http://127.0.0.1:8000/chat.html`

## 项目结构

- `main.py`：API 与页面路由
- `static/`：前端静态资源
- `deploy/`：部署示例配置
- `scripts/`：辅助脚本
- `data/`：运行时记忆数据

## 开发说明

- 前端请求会统一走配置化 API 地址
- 记忆数据按 `client_id` 组织
- 分析结果会尽量以自然中文输出，避免暴露原始 JSON
